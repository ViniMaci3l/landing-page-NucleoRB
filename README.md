# Landing Page Nucleo RB

Landing page premium em React + Vite para o Nucleo RB / Dra. Renata Bogea.

## Rodar localmente

```bash
npm install
npm run dev
```

URL padrao do Vite:

```text
http://127.0.0.1:5173/
```

## Validacao

```bash
npm run lint
npm run build
```

## Assets reais usados

- `/assets/logo.png`: header e rodape.
- `/assets/renata1.jpeg`: composicao visual da hero section.
- `/assets/renata2.jpeg`: secao Dra. Renata Bogea.
- `/assets/favicon.svg`: favicon do site.
- `/assets/icons.svg`: mantido em `public/assets`, sem uso forcado.
